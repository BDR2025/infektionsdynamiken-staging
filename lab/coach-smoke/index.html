Smoke Test / Ben global / Chloe lokal

Coach UI mit echtem KI-Durchstich

Diese Seite verbindet einen globalen Header-Coach mit einem lokalen Spot-Coach. Ben haengt oben rechts im Header, Chloe bleibt an einem konkreten Lernmoment.

Was du sofort testen kannst

  • Ben erklaert, wo du bist und was diese Seite leisten soll.
  • Chloe bleibt eng an einem lokalen Spot und gibt knappe Hilfestufen.
  • Beide Antworten laufen ueber denselben lokalen GitHub-Models-Proxy.

1 / Header

Ben sitzt oben rechts

Klick im Header auf den Ben-Coin. Gute Testfragen sind: Wo bin ich hier?, Was soll diese Seite tun? und Was ist mein naechster Schritt?.

npm run dev npm run coach:proxy

2 / Prompt-Schnitt

Guardrails vor Magie

Der Smoke Test prueft bewusst keine Super-KI. Wir testen erst Rollenstabilitaet, Kontexttreue und klare Grenzen: Ben ist global, Chloe lokal, beide bleiben knapp und erfinden nichts.

  • Ben antwortet deutsch und seitenorientiert.
  • Chloe antwortet englisch und spot-orientiert.
  • Der Proxy haelt Token und Modellwahl vom Frontend fern.

Was dieser Durchstich prueft

Wenn Ben im Header sauber erklaert, wo du gerade bist, und Chloe am Spot konkret bleibt, dann haben wir den eigentlichen Produktkern schon getroffen: globaler Kontext plus lokale Hilfe auf derselben technischen Schiene.

Danach koennen wir gezielt nachschaerfen: feinere Sprachprofile, mehr Spots, besseres Context Packing und spaeter echte Projekt- oder Seitenwissensquellen.
